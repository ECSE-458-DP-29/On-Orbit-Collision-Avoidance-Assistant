📖 About OOCAA

OOCAA is a specialized tool for aerospace professionals and operators managing satellite operations. It provides real-time analysis of conjunction data, probability of collision calculations, and risk assessment to support decision-making in conjunction assessment and avoidance maneuvering.

Key features include CDM ingestion, collision probability calculation using CARA MATLAB tools, event tracking, and comprehensive filtering and search capabilities.
