File Format Requirements

Your JSON file should contain CDM data in CCSDS format with the following key fields:

  • CCSDS_CDM_VERS: CDM format version (e.g., "1.0")
  • CREATION_DATE: ISO 8601 timestamp
  • TCA: Time of Closest Approach
  • MISS_DISTANCE: Minimum distance in km
  • SAT1_*: Object 1 parameters (position, velocity, covariance)
  • SAT2_*: Object 2 parameters (position, velocity, covariance)

The file can contain a single CDM object or an array of CDM objects.
